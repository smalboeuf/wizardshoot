# Wizard Shoot

Wizard Shoot is a simple 2D Top down game where you try and survive as many rounds as you can against the endless horde of monsters!

# Mechanics

- Top Down movement
- Enemy chasing
- Shooting
- Item Drops and buffs

## Stack

- Game Engine: Unity
- Pixel Art: Aseprite
